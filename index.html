Projects

Artefact Analysis Report

This project is a comprehensive analysis of a uPVC pipe sample intended for rainwater harvesting in coastal areas. The study evaluates the pipe's composition using FTIR spectroscopy and its mechanical properties (tensile, impact, flexural, and hardness) to ensure it meets durability requirements in environments with high UV exposure and salinity. The report concludes with recommendations for compositional and manufacturing adjustments to improve performance and reduce production costs

Failure Analysis Report

This project is a failure analysis performed on a mild steel sample that failed during a nick bend test. The investigation involved Optical Emission Spectrometry (OES), hardness testing, and microstructure analysis to determine the failure's root cause. The analysis concluded that the ductile failure was primarily due to manganese sulfide (MnS) inclusions, which facilitated microvoid nucleation and growth under stress.

Capstone Prorject Proposal + Report for Group-2

This project is a proposal for a capstone project focusing on a green pathway for producing energy by generating hydrogen from wastewater. The proposed method involves two main steps: first, using an adsorption mechanism to isolate heavy metals from the wastewater, and second, splitting the purified water into hydrogen gas using solar-powered electrolysis. The project aims to make productive use of contaminated industrial wastewater while producing clean energy

Investigation of Microstructure and Mechanical Properties of Welded Joint

This experiment focused on investigating the microstructure and mechanical properties of a welded joint, specifically one prepared using Gas Tungsten Arc Welding (GTAW). The project included macro and micro analysis to observe microstructure control and evaluate the Hall-Petch effect based on grain size variation. Mechanical testing was also performed, including transverse tensile, nick-break, and bend tests on a butt-welded pipe joint.

MME 308 Task

This document presents results for two distinct laboratory tasks.
